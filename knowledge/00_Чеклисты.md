# Чеклисты
Из регламента документооборота Коченевских бюро.

---

17. ЧЕК-ЛИСТЫ
Этот раздел — короткая шпаргалка для комплектатора.
Если нет времени перечитывать всю инструкцию, проверь себя по нужному чек-листу перед отправкой счета, передачей товара, оформлением документов или сдачей документов в архив.
17.1. Чек-лист перед отправкой счета клиенту
Перед отправкой счета клиенту проверь:
выбран правильный клиент;
указана корректная сумма;
позиции, количество и стоимость совпадают с согласованным заказом;
в счете нет закупочной цены;
в счете нет внутренней информации, дилерской скидки или бонуса;
НДС отражен корректно;
доставка до объекта указана / прокомментирована корректно;
срок действия счета понятен клиенту;
если клиент физлицо — МД предупрежден, что может поступить оплата и потребуется чек;
счет скачан в PDF;
для счета включены подпись, печать и QR-код;
файл проверен перед отправкой.
17.2. Чек-лист после оплаты клиентом
После оплаты клиентом проверь:
получено подтверждение оплаты от клиента, если клиент его прислал;
МД передана информация по оплате;
МД передан счет для клиента;
МД передан счет поставщика, если поставщика нужно оплатить;
если клиент физлицо — МД отдельно предупрежден, что нужно пробить чек;
поступление денег проверено;
чек пробит, если он нужен;
счет поставщика оплачен, если это дилерская продажа;
поставщик получил подтверждение оплаты;
поставщик подтвердил срок отгрузки / поставки;
информация внесена в таблицу Документооборот Kochenevskikh Design;
информация внесена в таблицу Учет комплектации.
17.3. Чек-лист перед приемкой товара от поставщика
Перед приемкой товара проверь:
понятно, на кого оформлен заказ: на клиента или на ИП Коченевских С.С.;
понятно, кто принимает товар;
известен адрес доставки;
известны количество мест / состав заказа;
есть счет / спецификация / заказ, с которыми можно свериться;
подготовлен телефон для фотофиксации;
если товар на ИП Коченевских С.С. — комплектатор не подписывает документы поставщика от имени ИП;
если поставщик привез бумажные документы — их нужно забрать на подпись руководителю, если поставщик готов оставить документы;
при повреждениях / расхождениях — сразу фото и сообщение руководителю отдела реализации.
17.4. Чек-лист при передаче товара клиенту / на объект
Перед передачей товара проверь:
понятно, кто принимает товар;
если принимает не клиент лично — есть письменное подтверждение от клиента;
подготовлена накладная / УПД / акт / расписка;
если клиент физлицо — чек уже выдан или подготовлен к передаче;
товар сверен по количеству мест;
упаковка сфотографирована;
при передаче получена подпись принимающего лица;
наш экземпляр документа забран обратно;
если товар оставляется без приемки — есть письменное разрешение клиента;
если товар оставляется без приемки — сделаны фото / видео размещения товара;
клиенту отправлены фото / видео;
получено подтверждение клиента: «товар принят» / «размещение подтверждаю»;
информация внесена в таблицу;
документы сохранены в папку объекта.
17.5. Чек-лист перед созданием накладной / УПД в Эльбе
Перед созданием накладной / УПД проверь:
документ создается на основании правильного счета;
клиент выбран корректно;
номер документа взят из таблицы Документооборот Kochenevskikh Design;
дата документа соответствует дате отгрузки / передачи товара;
позиции подтянулись корректно;
количество и стоимость совпадают со счетом;
НДС отражен корректно;
основание подтянулось корректно;
в документе нет закупочной цены;
в документе нет внутренней информации;
для бумажной накладной / УПД подпись и печать из Эльбы выключены;
документ передан на подпись руководителю / ИП Коченевских С.С. или МД для отправки через ЭДО.
17.6. Чек-лист по дизайнерскому бонусу
Перед запуском работы по дизайнерскому бонусу проверь:
поставщик подтвердил, что бонус предусмотрен;
понятен размер бонуса;
понятно, от какой суммы считается бонус;
понятно, когда бонус выплачивается;
понятно, какие документы нужны для выплаты;
уточнено, есть ли у поставщика договор;
уточнено, работает ли поставщик через ЭДО;
договор проверен на базовую адекватность;
в договоре корректно отражена наша СНО: УСН 7% + НДС 5%;
документы переданы МД;
после отправки документов поставщику комплектатор контролирует оплату бонуса;
информация внесена в таблицу  Учет комплектации.
17.7. Чек-лист по дилерскому договору
Перед передачей дилерского договора МД проверь:
договор оформляется с ИП Коченевских С.С.;
понятен размер дилерской скидки;
понятно, от какой цены считается скидка;
нет обязательного плана закупок без согласования;
нет штрафа за невыполнение плана продаж;
нет обязанности держать складской остаток;
понятны условия оплаты;
понятны условия поставки и доставки;
понятны условия возврата / замены / брака;
в договоре корректно отражена наша СНО: УСН 7% + НДС 5%;
понятно, работает ли поставщик через ЭДО;
МД передан договор, наименование поставщика, ИНН, оператор ЭДО и комментарий по условиям.
17.8. Чек-лист перед передачей документов в архив
Перед передачей документов МД проверь:
документы разложены по клиентам / объектам;
у каждого клиента / объекта есть свое отделение в папке;
дилерские документы и документы поставщиков лежат отдельно;
подписанные документы не лежат вперемешку с неподписанными;
документы поставщика с закупочной ценой не могут случайно попасть к клиенту;
все накладные / УПД / акты подписаны;
если клиент физлицо — чек выдан;
по нестандартным передачам товара сохранены фото / видео и переписка;
информация внесена в таблицу Документооборот Kochenevskikh Design;
информация внесена в таблицу  Учет комплектации;
спорные или непонятные документы отдельно показаны МД.
